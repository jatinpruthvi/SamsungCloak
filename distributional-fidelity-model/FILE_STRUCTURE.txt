─── STATISTICAL REALISM HOOKS ────────────────────────────────────────

1. Conversion Rate Variance
   Distribution: Binomial
   Mathematical: P(X = k) = C(n, k) * p^k * (1-p)^(n-k)
   Key Features:
   - Simulate success/failure fluctuations
   - Population-level conversion variance
   - Temporal variance (rates change over time)
   - Expected value and variance calculations

2. Engagement Distribution
   Distribution: Pareto (Power Law)
   Mathematical: f(x) = (α * x_m^α) / (x^(α+1)) for x ≥ x_m
   Key Features:
   - 80/20 rule implementation
   - Power user identification
   - Gini coefficient for inequality
   - Segmented population generation

3. Action Frequency Distribution
   Distribution: Poisson Process
   Mathematical: P(X = k) = (λ^k * e^(-λ)) / k!
   Inter-arrival: T ~ Exponential(λ)
   Key Features:
   - Event timing and clustering
   - Message burst simulation
   - Non-homogeneous time-varying rates
   - Multi-user parallel simulation

4. Long-Tail Behavior Modeling
   Distribution: Heavy-tailed mixtures
   Methods: Extreme value theory, IQR detection
   Key Features:
   - Rare, high-intensity outlier sessions
   - Multiple outlier types (CPU, Memory, Network, Duration)
   - Stress test scenarios
   - Statistical outlier detection

5. Non-Uniform Sampling
   Distribution: Gaussian (Normal)
   Mathematical: f(x) = (1 / (σ * √(2π))) * e^(-((x-μ)²) / (2σ²))
   Key Features:
   - Box-Muller transform for sampling
   - Constrained and integer sampling
   - Mixed demographic segments
   - Correlated and autocorrelated samples

─── MATHEMATICAL FOUNDATIONS ──────────────────────────────────────

Probability Density Functions (PDF):
  Binomial: P(X=k) = C(n,k) * p^k * (1-p)^(n-k)
  Pareto: f(x) = (α * x_m^α) / (x^(α+1))
  Poisson: P(X=k) = (λ^k * e^(-λ)) / k!
  Normal: f(x) = (1 / (σ√(2π))) * e^(-((x-μ)²)/(2σ²))

Key Properties:
  Binomial: Discrete, fixed trials, independent
  Pareto: Heavy-tailed, power law, 80/20
  Poisson: Event counting, exponential inter-arrival
  Normal: Symmetric, bell curve, 68-95-99.7 rule

Statistical Concepts:
  - CDF (Cumulative Distribution Function)
  - Inverse CDF (Quantile Function)
  - Gini Coefficient (Inequality measure)
  - IQR (Interquartile Range) for outliers
  - Box-Muller Transform (Normal sampling)
  - Inverse Transform Sampling (General method)
